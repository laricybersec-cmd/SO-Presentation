# my-SO-website
Aplicação web interativa para a disciplina de Sistemas Operacionais, focada em UI/UX moderna com HTML5, CSS3 e Flexbox. O projeto explora a evolução tecnológica, do pioneirismo multitarefa do AmigaOS à segurança robusta do Kali Linux, unindo design narrativo e experiência multimídia para analisar a infraestrutura digital global.
